A Fabric mod for Minecraft 1.20.4 that makes campfire creation more immersive and interactive! Instead of instant crafting, experience a satisfying multi-step process that feels natural and rewarding.

## ‚ú® Features

### ü™ì The Chopping Stump
- Craft a **Stump** from 3 logs in a row
- Place any log on the stump by right-clicking
- Right-click again to **chop** the log into 4 firewood pieces

### ü™µ Firewood Stacking
- Place firewood on the ground
- Stack up to **4 firewood pieces** - they arrange like real campfire logs
- Uses authentic campfire log dimensions and textures

### üî• Lighting the Fire
- Once you have 4 firewood stacked, right-click with **coal or charcoal**
- This creates an **Unlit Campfire** ready to be ignited
- Use **Flint and Steel** (or Fire Charge) to light it
- Transforms into a vanilla campfire!

## üì¶ Installation

1. Install [Fabric Loader](https://fabricmc.net/use/installer/) for Minecraft 1.20.4
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download the mod JAR from [Releases](https://github.com/sahil-tgs/Immersive-Campfire/releases)
4. Place in your `.minecraft/mods` folder
5. Launch the game!

## üîß Building from Source

```bash
# Clone the repository
git clone https://github.com/sahil-tgs/Immersive-Campfire.git
cd Immersive-Campfire

# Build the mod
./gradlew build

# The JAR will be in build/libs/
```
